# Fracture in Rotating Structures: Failure Analysis

This repository provides an interactive analysis of fracture mechanics in rotating structures using **dimensionless Stress Intensity Factors (SIFs)**.
The analysis is implemented as a **marimo notebook**, which is automatically deployed to [GitHub Pages](https://floiseau.github.io/fracture_in_rotating_structures_failure_analysis).

## How to Use

### Online Version
Visit the [deployed notebook](https://floiseau.github.io/fracture_in_rotating_structures_failure_analysis).

### Local Version
1. Clone the repository:
   ```bash
   git clone https://github.com/floiseau/fracture_in_rotating_structures_failure_analysis.git
   cd fracture_in_rotating_structures_failure_analysis
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the notebook locally:
   ```bash
   marimo edit failure_in_rotating_structures_failure_analysis_nb.py
   ```
